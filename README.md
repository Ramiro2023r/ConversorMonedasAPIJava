# ConversorDeMonedasAPIAlura
Proyecto de conversor de monedas consumiendo una API que nos da el valor actual de las monedas que queremos convertir.
# 💱 Conversor de Monedas - Spring Boot

Proyecto desarrollado en Java con Spring Boot que consume la API ExchangeRate-API para realizar conversiones de monedas en tiempo real.

## 🚀 Tecnologías
- Java 17
- Spring Boot
- Gson
- HttpClient
- ExchangeRate-API

## 📌 Funcionalidades
- Conversión USD, ARS, BRL, COP
- Consumo de API externa
- Parseo JSON con Gson
- Menú interactivo en consola

## ▶️ Cómo ejecutar
1. Clonar el proyecto
2. Agregar tu API KEY en ExchangeRateService
3. Ejecutar ConversorMonedaApplication
